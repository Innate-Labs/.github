# Security Policy · 安全政策

## Reporting a vulnerability · 上报漏洞

If you discover a security vulnerability in any Innate Labs project, **please do not open a public issue.**
如果你在 Innate Labs 的任何项目中发现安全漏洞，**请不要公开提 issue。**

Instead, report it privately by emailing **huboachg@gmail.com** with:
请通过邮件私下上报至 **huboachg@gmail.com**，并附上：

- A description of the vulnerability · 漏洞描述
- Steps to reproduce · 复现步骤
- Potential impact · 可能影响

We will acknowledge your report as soon as possible and keep you updated on the fix.
我们会尽快确认，并向你同步修复进展。
